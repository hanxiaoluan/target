# Target
大目标化成小目标，小目标行动起来！

## 怎么使用
```
运行服务端
1. cd server
2. cp database/db-dev.json db.json
3. npm install /  yarn
4. npm run start / yarn start
```
运行前端
```
1. cd frontend
2. npm install /  yarn
3. npm run dev / yarn dev
```
运行测试
```
1. cd test
2. npm install /  yarn
3. npm run start / yarn start
```
